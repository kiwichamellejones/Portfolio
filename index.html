DevDiary

Description: DevDiary is a responsive personal blog platform designed for developers and tech enthusiasts to share their knowledge, insights, and experiences. The platform features an easy-to-use content management system for creating and managing blog posts, complete with categories, tags, and a comments section for reader interaction.

Purpose: The primary purpose of DevDiary is to provide a space for developers to document their learning journey, share valuable resources, and engage with a community of like-minded individuals. It also serves as a personal portfolio to showcase writing skills and technical expertise.

Technologies Used:

  • HTML
  • CSS (Bootstrap)
  • JavaScript
  • Jekyll (Static Site Generator)
  • Netlify CMS (Content Management System)

WeatherWise

Description: WeatherWise is a weather forecast dashboard that provides real-time weather updates and forecasts for multiple locations. Users can search for current weather conditions and 5-day forecasts for any city worldwide, complete with detailed weather information and intuitive icons.

Purpose: The primary purpose of WeatherWise is to offer users a reliable and easy-to-use tool for accessing weather information. It aims to provide accurate weather forecasts to help users plan their activities and stay informed about changing weather conditions.

Technologies Used:

  • HTML
  • CSS
  • JavaScript
  • React
  • OpenWeatherMap API
  • Netlify (Deployment)

BizConnect CRM

Description: BizConnect CRM is an online customer relationship management system designed to help businesses manage customer interactions, track sales, and organize customer data efficiently. The application includes secure login authentication, allowing only authorized users to access and manage customer information.

Purpose: The primary purpose of BizConnect CRM is to provide businesses with a comprehensive tool for managing customer relationships, enhancing customer service, and streamlining business operations. It enables businesses to keep track of customer interactions, monitor sales activities, and improve overall productivity.

Technologies Used:

  • HTML
  • CSS
  • JavaScript (React)
  • Node.js (Express.js)
  • MongoDB
  • JWT (JSON Web Tokens) for Authentication
  • Heroku (Deployment)
